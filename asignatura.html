JAVA

Descripcion de la asignatura

Curso 2024/2025

Java es un lenguaje de programación y una plataforma informática que fue comercializada por primera vez en 1995 por Sun Microsystems Microsystems, en esta asignatura se impartirán los conocimientos desde un nivel básico hasta el nivel más profesional.

Temario

Tema 1 Introducción

En este tema introductorio comenzaremos con los tipos de proyectos y aplicaciones que se pueden desarrollar en java y los entornos de desarrollo que existen, ademas de aprender a crear un proyecto desde cero.

30% completado

Tema 2 Funciones y estructuras de control

En este tema trabajaremos con las estructuras de control como if, for y while ademas aprenderemos a optimizar el codigo mediante el uso de funciones.

50% completado

Tema 3 Clases y objetos

En este tema aprenderemos a crear clases y objetos en java.

0% completado
